This analysis uses the [Higher Education Students Performance Evaluation](https://www.kaggle.com/datasets/mariazhokhova/higher-education-students-performance-evaluation) data set from Kaggle, from the Faculty of Engineering and Faculty of Educational Sciences students in 2019 to explore the **demographic**, **socioeconomic**, and **behavioral** factors influencing academic success, combining descriptive, diagnostic, and predictive approaches to generate actionable insights
